# RMT（若梦兔）
**Read this in other languages: [English](README.en.md).**

## 介绍
RMT（若梦兔）是一个基于 AutoHotkey v2 实现的免费开源按键工具，支持按键宏、鼠标宏、手柄宏、图片识别、颜色识别、文本识别等多种功能。
